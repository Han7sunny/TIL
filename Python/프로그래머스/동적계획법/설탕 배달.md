# 설탕 배달
#### 백준 2839번
[백준 | 설탕 배달](https://www.acmicpc.net/problem/2839)
### 풀이
```python
from sys import stdin

n = int(stdin.readline()) # 5000
answer = 0
while n >= 0:
    if n % 5 == 0:
        answer += n // 5
        print(answer)
        break
    n -= 3
    answer += 1

else: # while 탈출 (n 음수)
    print(-1)

```
